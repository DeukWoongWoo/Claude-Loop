# CLAUDE.md Validator

Analyze CLAUDE.md files against best practices and provide actionable improvements with specific line references.

## Usage

Invoke this skill when you need to validate or improve a CLAUDE.md file.

**Trigger phrases:**
- "Validate my CLAUDE.md"
- "Review CLAUDE.md file"
- "Check CLAUDE.md quality"
- "Improve my CLAUDE.md"

## Description

This skill validates CLAUDE.md files against established best practices to ensure they are token-efficient, well-structured, and contain project-specific information that helps Claude understand your codebase.

**Key capabilities:**
- Validates structure (line count, format, headers)
- Checks content quality (project-specific info, tech stack, file locations)
- Identifies anti-patterns (narrative style, inline details, redundant info)
- Provides line-specific suggestions for improvement

## Workflow

When invoked, the skill:

1. **Read**: Reads the target CLAUDE.md file
2. **Analyze**: Checks against validation criteria (structure, content quality, format)
3. **Report**: Generates findings with specific line references
4. **Suggest**: Provides concrete improvement recommendations

**Output**: A validation report containing:
- Summary with total lines and pass/fail status
- Issues found with line numbers and descriptions
- Priority recommendations for improvement

## Prerequisites

No special setup required.
